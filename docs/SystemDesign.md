# Functional Requirements

Our application should allow users to:

1. Authentication:
    Register
    Login
    Forgot Password (later)
    Logout

2. Dashboard:

    Show:
        Total Notes
        Total Quizzes
        Average Score
        Study Time
        AI Usage

3. Notes:

    Upload:
        PDF
        DOCX
        PPTX

    Operations:
        View
        Delete
        Rename
        Download

4. AI Features:

    Generate:
        Summary
        Flashcards
        MCQs
        Short Questions
        Important Topics
        Keywords
        Difficulty Level

5. AI Chat:

    User asks:---Explain JVM.

    AI searches uploaded notes
            ↓
    Returns answer
            ↓
    Shows source page numbers

6. Analytics:

    Charts:
        Weekly study
        Monthly study
        Quiz score trend
        Upload trend

7. Profile:
    Name
    Email
    Password
    Theme
    Profile Picture
